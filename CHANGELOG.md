# CHANGELOG

## v1.1.1 - 2026-07-16

### Changed

* TOEICで一般的な用法を優先し、185語の日本語訳・品詞を改善
* 古い語義、固有名詞、まれな意味が優先されていた単語を現代的な意味へ修正
* TOEICで重要な多義語やビジネス用法を追加

### Fixed

* 誤訳、不自然な日本語、品詞と意味の不一致を修正
* 壊れた括弧や途中で切れていた意味を修正

## v1.1.0 - 2026-07-12

### Added

* 忘却曲線に基づく自動復習スケジュール
* 単語ごとの記憶強度管理
* 短期学習・再学習フェーズ
* 忘却回数の記録
* 回答後の次回復習予定表示
* 学習履歴のリセット機能
* 既存学習データの自動移行

### Changed

* 固定間隔方式を、記憶状態に応じた復習方式へ変更
* 復習単語の出題順を期限超過率に基づく方式へ改善
* 学習計画を固定倍率ではなく、実際の復習予定日時から計算
* 習熟レベルと学習状態を編集画面で読み取り専用化
* 今日の復習回答数と復習予定語数の表示を分離

### Fixed

* 不正な日時データで編集画面が停止する可能性を修正
* 既存データ移行時に次回復習日時が変更されないよう対応

## v1.0.0 - 2026-07-12

### Added

* Git導入
* CSVインポート
* CSVエクスポート
* 約1540語登録

### Changed

* CSV形式を新仕様へ変更

### Fixed
