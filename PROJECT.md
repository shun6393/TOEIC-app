## 概要

## 技術

* CSS
* JavaScript
* Local Storage
* Git

## 実装済み

* Git導入
* CSVインポート
* CSVエクスポート
* 学習履歴
* 目標点数設定
* 単語検索
* 約1540語登録済み
* 忘却曲線に基づく自動復習
* 記憶強度に応じた次回復習日時の計算
* 短期学習・再学習
* 復習単語の優先出題
* 既存学習データの自動移行

## 復習方式

単語ごとの記憶強度と回答結果に基づいて、次回復習日時を自動で計算する。

回答は以下の3段階で行う。

* 知らない
* 微妙
* 知ってる

新しく学習する単語や忘れた単語は短期学習を行い、その後は記憶状態に応じて復習間隔を調整する。

### ローカル実行

開発時はローカルHTTPサーバーを使用する。

例
- VS Code Live Server
- `python -m http.server`
- http://localhost:8000で開く
※ `index.html` を `file://` で直接開くと、Fetch APIによる標準単語CSVの自動読み込みが動作しない。
